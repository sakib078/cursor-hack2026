
## Product Summary

Afterwords is a grief-support experience that lets a user upload real conversations and voice recordings from someone they have lost, then interact with an AI-powered memorial that responds in a familiar voice and style. The core value is emotional continuity: helping the user feel heard, remember accurately, and say what was left unsaid.

This PRD is scoped to the 2-hour hackathon build plan and focuses on a demoable end-to-end experience rather than a production-ready system.

## Problem Statement

People carrying grief, breakup pain, or unresolved loss often want one more conversation with the person they miss. Existing chatbots feel generic and detached. Afterwords aims to create a more personal, emotionally resonant interaction by grounding responses in the person’s real language, tone, and voice.

## Goals

- Let a user create a memorial session from uploaded chat history and optional voice/audio files.
- Build a strong persona prompt from the person’s real messages so the AI sounds recognizably like them.
- Provide a polished chat experience where the user can converse with the memorial.
- Support a coping or support mode that shifts the experience toward comfort and grounding.
- Deliver a reliable demo path within 2 hours.
